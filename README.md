# Visuva.Site - 图片资源管理与共享平台

## 一、问题描述

本项目旨在为用户提供一个高效、安全、易用的图片资源管理与共享平台，满足企业及个人在图像资料存储、分类、检索、分享和分析等方面的多样化需求。系统主要面向企业管理员、企业成员、普通用户及平台管理员，支持多角色协同操作。

系统共提供三大功能模块：
1. **团队共享图库**：用于企业内部团队协作，实现团建活动照片、设计素材、敲钟记录等集体性图像资源的集中管理与共享。
2. **私有图库**：支持用户创建个人专属空间，用于存放私人照片、艺术作品、学习资料等敏感或个性化内容，并具备权限控制与数据分析能力。
3. **图库平台**：作为开放型公共平台，提供表情包、壁纸、设计素材等通用资源，供注册用户上传、下载、浏览与使用，同时由管理员进行内容审核与管理。

该软件的核心功能包括：用户注册登录、图片上传/导入、批量处理、编辑管理、搜索查询、浏览分享、下载使用、权限分配、空间分析以及内容审核等。系统支持多类型图像资源的分类存储（如团建相册、艺术与运动、学习资料、表情包等），并能对图片数据进行统计分析，提升资源利用效率与用户体验。

## 二、系统设计

该图库管理系统可划分为以下三个主要模块：

### 团队共享图库模块
由企业管理员负责开通空间、邀请/管理成员，并进行空间分析；企业成员可上传/导入图片，进行管理与编辑，支持搜索、浏览与分享图片。该模块包含“团建相册”、“设计素材”、“敲钟记录”三个子分类，实现团队协作场景下的图像资源共享与统一管理。

### 私有图库模块
管理员可开通并管理所有私有空间，进行全局分析；用户可创建个人空间，上传/导入图片，进行管理、编辑、搜索、浏览与分享，并支持空间使用情况分析。该模块包含“个人照片”、“艺术与运动”、“学习资料”等分类，保障用户隐私与个性化存储需求。

### 图库平台模块
支持管理员注册登录后上传/批量导入图片，并对用户上传的内容进行审核；普通用户需注册登录后方可上传/导入图片，通过搜索查找所需资源，并进行浏览与下载。平台内设“表情包”、“壁纸”、“设计素材”等公共分类，形成开放共享的图像资源池。

当运行该系统时，应首先显示主菜单界面，用户可根据身份选择进入对应的三大模块之一：团队共享图库、私有图库 或 图库平台，系统将根据用户角色权限自动跳转至相应功能入口，确保操作的安全性与便捷性。

## 三、项目功能梳理

### 第一阶段 - 公共图库平台

#### 用户模块
- 用户登录
- 用户注册
- 用户注销
- 用户权限控制
- 【管理员】管理用户

#### 图片模块
- 【管理员】上传创建图片
- 【管理员】图片信息编辑（标签 / 分类等）
- 【管理员】管理图片
- 查看和搜索图片列表
- 查看图片详情（进入图片详情页）
- 图片下载
- 用户上传创建图片
- 【管理员】审核图片
- 导入图片
  - 通过 URL 导入图片
  - 【管理员】批量抓取和创建图片
- 【优化】图片查询优化 - 分布式缓存、本地缓存、多级缓存
- 【优化】图片上传优化 - 压缩、秒传、分片上传、断点续传
- 【优化】图片加载优化 - 懒加载、缩略图、CDN 加速、浏览器缓存
- 【优化】图片存储优化 - 降频存储（冷热数据分离）、清理策略

### 第二阶段 - 用户私有图库

#### 空间模块
- 【管理员】管理空间
- 用户开通私有空间
- 私有空间权限控制
- 空间级别和限额控制
- 空间图库分析
  - 用户空间图库分析
  - 【管理员】全空间分析

#### 图片模块
- 图片搜索
  - 基础属性搜索
  - 以图搜图
  - 颜色搜索
- 图片分享
  - 链接分享
  - 扫码分享
- 图片批量管理
  - 批量修改信息
  - 批量重命名
- 图片编辑
  - 基础图片编辑
  - AI 图片编辑

### 第三阶段 - 团队共享图库

#### 空间模块
- 创建团队共享空间
- 空间成员管理
  - 成员邀请
  - 设置权限
- 空间成员权限控制

#### 图片模块
- 图片协同编辑

- COS 对象存储

- Sa-Token 权限控制

- Ant Design Vue 组件库
- Axios 请求库
